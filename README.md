# Wwwtxt

## Installation

```
gem install wwwtxt
```

## Usage

(Local)

```
bundle exec bin/wwwtxt -o "~/texts" get https://raw.githubusercontent.com/ryanmuller/www.txt/master/README.md
```

## Running tests

```
ruby test/wwwtxt/test_page.rb
```
